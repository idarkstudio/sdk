# dfx schema

<MarkdownChipRow labels={["Referencia"]} />

El comando `dfx schema` imprime el esquema para `dfx.json`.

## Uso básico

```bash
dfx schema [option] [flag]
```

## Opciones

Puedes usar la siguiente opción con el comando `dfx schema`:

| Opción                | Descripción                                                                         |
| --------------------- | ----------------------------------------------------------------------------------- |
| `--for <which>`       | Muestra el esquema para un archivo JSON específico. (Por defecto: `dfx`).           |
| `--outfile <outfile>` | Especifica un archivo de salida para el esquema en lugar de imprimirlo en `stdout`. |

El parámetro pasado a `--for` puede ser uno de los siguientes:

| Opción `<for>`           | Corresponde a                                                                                                        |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| `dfx`                    | `dfx.json`                                                                                                           |
| `networks`               | `networks.json`                                                                                                      |
| `dfx-metadata`           | Metadatos relacionados con `dfx` almacenados en el canister y accesibles con `dfx canister metadata <canister> dfx`. |
| `extension-manifest`     | `extension.json` [ejemplo][example-extension-json].                                                                  |
| `extension-dependencies` | `dependencies.json` [ejemplo][example-dependencies-json].                                                            |

## Ejemplos

Puedes imprimir el esquema para `dfx.json` ejecutando el siguiente comando:

```bash
dfx schema
```

Puedes imprimir el esquema para `networks.json` ejecutando el siguiente comando:

```bash
dfx schema --for networks
```

Puedes imprimir el esquema para el JSON de metadatos "dfx" (que contiene campos
como "pullable", "tech_stack", etc) ejecutando el siguiente comando:

```bash
dfx schema --for dfx-metadata
```

Si deseas escribir el esquema para `dfx.json` en `path/to/file/schema.json`,
puedes hacerlo ejecutando el siguiente comando:

```bash
dfx schema --outfile path/to/file/schema.json
```
